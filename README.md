# Linh-first-dataEngineer

Repo này dùng để ghi lại quá trình học GitHub, Python, SQL và nền tảng Data Engineer từ con số 0.

## Mục tiêu

- Làm quen với GitHub
- Lưu bài học và code cá nhân
- Học Python cơ bản
- Học SQL cơ bản
- Làm việc với file CSV
- Tạo mini project ETL

## Nội dung học

1. Git/GitHub cơ bản
2. Python cơ bản
3. SQL cơ bản
4. CSV và xử lý dữ liệu
5. Mini ETL project

## Ghi chú

Đây là repo học tập cá nhân, dùng để luyện tập và xây dựng nền tảng Data Engineer.
